Writing Beautiful Docs

Docs doesn't have to be hard. Especially with PowerShell. Combine PlatyPS with mkdocs, and all you need to worry about is your comment-based help. This documenation was automatically generated in a build pipeline for my talk "Generating Beautiful Documentation" for the Raleigh PowerShell User Group, held on 18 August 2021.

Use this code!

All the code available in the repository at https://github.com/steviecoaster/BeautifulDocs can be used as a baseline to implement this in your organization, or in your personal projects. Enjoy!
